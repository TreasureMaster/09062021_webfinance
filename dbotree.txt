Balance                 данные для отчетов по балансу предприятий
Clients                 информация о клиентах (контрагентах, лизингополучателях)
Contract                полная информация о договорах
FinanceResults          данные для отчетов по финансовым результатам предприятий
Group_of_companies      список компаний в группе компаний (вроде не используется)
Incorporation           список организационно-правовых форм предприятий
MutualTurnovers         взаимные обороты предприятий (для коррекции отчетности по группе предприятий)
ParentCovenants         исходные данные для ковенант материнской компании
Payments                даты исполненных платежей по каждому договору
Product                 описание техники
ProductCategory         описание категорий техники
Regions                 список регионов, в которых зарегистрированы клиенты
ReportsTitle            просто список заголовков таблиц для отчетностей по балансу и финансовым результатам
Storage                 склад
SubsidiaryCovenants     исходные данные для ковенант дочерней компании
